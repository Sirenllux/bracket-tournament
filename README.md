Bracket

An image-based tournament tool. Upload images of your favorite characters, foods, movie posters, or anything else, then pick one of two until only one remains.

How to use

- Choose a round size
- Upload image files or a folder
- Click start
- For each match, click the one you prefer

Features

- Korean / English / Japanese language support
- Dark/light theme
- Automatic byes and preliminary matches
- Folder structure preserved on upload
- Mobile responsive
